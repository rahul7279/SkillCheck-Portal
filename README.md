## 🎯 Project Overview
SkillCheck is a lightweight, frontend-only assessment tool designed for hackathons and educational purposes. It promotes fair self-evaluation and digital exam discipline without requiring any backend database.

## ✨ Key Features

* 🧠 **Smart Performance Analysis (Rule-based):** Evaluates user scores across multiple subjects and provides actionable feedback (e.g., Deep Study vs. Revision).
* 🛡️ **Exam Security (Anti-Cheating):** Uses DOM Visibility API to detect tab switching. The test auto-submits after 3 warnings.
* 💾 **Offline Data Persistence:** Saves user progress, history, and warning logs locally using HTML5 `localStorage`.
* 📊 **Dynamic Results:** Displays Accuracy, Percentile, Cutoff tracking, and a dynamic Circular Progress Bar.
* 🎨 **Glassmorphism UI:** Clean, responsive, and modern user interface.
* ⌨️ **Accessibility:** Supports full keyboard navigation (Arrow keys).
